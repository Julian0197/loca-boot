# loca

## install
```bash
yarn

# 高版本暂时会导致发包失败，这个版本目前比较稳定。
yarn global add lerna@3.22.1
```

## 打包
```bash
# build
yarn pkgs boot
# publish
yarn pub boot
# 更新版本号
yarn v boot 
```
